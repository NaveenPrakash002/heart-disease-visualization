# heart-disease-visualization
## Objective
Analyze and visualize the Heart Disease dataset to uncover 
patterns, trends and relationships between key health factors 
like age, cholesterol, gender and heart disease presence.

## Dataset
- Source: Kaggle
- Features: Age, Gender, Chest Pain Type, Cholesterol, 
  Resting BP, Max Heart Rate, Heart Disease

## Tools Used
- Python
- pandas
- matplotlib
- seaborn
- Google Colab

## Visualizations Used
- **DistPlot** — Age distribution analysis
- **Pie Charts** — Categorical columns like Gender and Chest Pain Type
- **Violin Plot** — Distribution of Age, Cholesterol across groups
- **Heatmap** — Correlation between numerical features
- **Pair Plot** — Correlation across all numerical columns

## Key Findings
- Majority of individuals are aged between 40 to 70
- Males have higher frequency of heart disease than females
- Dataset is skewed with higher number of male individuals
- Age and Max Heart Rate show a negative correlation
- Cholesterol and Resting BP show relationships with heart disease
